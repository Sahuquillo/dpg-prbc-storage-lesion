# 2,3-DPG and pH in Stored Red Blood Cells — Reproducible Analysis

## Citation

Sahuquillo J, Peris M, Poca MA, Pons V, Rierola M. Rapid Depletion of
2,3-DPG During Red Blood Cell Storage Assessed by Liquid
Chromatography–Mass Spectrometry (LC–MS): Implications for Oxygen
Delivery in Neurocritical Patients. *Journal of Neurotrauma* (under
review, 2026).

Zenodo repository (data + code): [DOI to be provided at final submission]

## Overview

This repository contains the data preparation and statistical analysis
code supporting the above manuscript. We characterized the temporal
dynamics of 2,3-diphosphoglycerate (2,3-DPG) depletion and metabolic
acidification in 24 leukodepleted packed red blood cell (pRBC) units
stored under standard blood bank conditions (days 3–42). 2,3-DPG was
quantified by LC–MS; pH was measured in parallel at each time point.
Statistical analysis used linear mixed-effects models (LMMs) with random
intercepts for pRBC unit, fitted in R using lme4/lmerTest.

## Dataset description

### dpg_data.rds / dpg_raw.xlsx
- 24 leukodepleted pRBC units (rows = storage days, columns = units)
- Variable `day`: storage day (3, 7, 10, 14, 17)
- Remaining variables: 2,3-DPG concentration per unit (µmol/g Hb)
- Values below the LC–MS detection limit (after day 17) are coded as `NA`
- Donor identifiers have been removed; units are identified by anonymous
  bag codes only

### ph_data.rds / pH_raw.xlsx
- 24 leukodepleted pRBC units across 11 time points (days 3–42)
- Variable `day`: storage day
- Remaining variables: pH per unit (measured at room temperature, 20–23 °C)

## How to reproduce the analysis

### Requirements
- R version 4.5.2
- RStudio version 2026.01.2
- Quarto (included with RStudio 2026.01.2)

### Required R packages

```r
install.packages(c(
  "broom", "emmeans", "flextable", "ggthemes", "here",
  "janitor", "lme4", "lmerTest", "performance", "psych",
  "readxl", "rstatix", "see", "tidyverse", "writexl"
))
```

### Steps

1. Clone or download this repository
2. Open the project in RStudio
3. To reproduce data preparation (requires raw Excel files):
   - Open `01_data_preparation.qmd`
   - Click **Render** or run `quarto render 01_data_preparation.qmd`
   - This generates `dpg_data.rds` and `ph_data.rds` in `data/`
4. To reproduce the analysis:
   - Open `02_analysis.qmd`
   - Click **Render** or run `quarto render 02_analysis.qmd`
   - Rendered output saved to `output/02_analysis.html`

> **Note:** Step 3 can be skipped if the `.rds` files are already present
> in the `data/` folder, as they are deposited in this repository.

### Session information
Full session information (R version, package versions, platform) is
included at the end of `02_analysis.qmd`.
